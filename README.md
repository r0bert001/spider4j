# Spider4j
> Spider4j is a simple data spider program，this is demo for learning.

### runtime requirement
- Java version：jdk-14
### tech stack
- dependency management：maven 3.6.2
- based on ：Spring boot 2.3.2.RELEASE

### target
learn java multi thread knowlege

### modules
- URL queue manager
- URL parser
- Image downloader
- Page downloader
- HTML parser
